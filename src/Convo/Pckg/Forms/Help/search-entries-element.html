Search Entry Element

Searches entries by a given filter.

Parameters:

  • Context ID - ID of the referenced Form context
  • Search - The value that represents search. It is highly dependent on referenced form context.
  • Return Variable Name - Default status, name of the variable that contains additional information

Result variable fields

Fields for property Result Variable Name
Field Description
result Array of entries found
count Total count of found results
value Single entry. Available only in the Single flow

Flows:

  • Multiple flow - will be executed if the requested search has found multiple entries
  • Single flow - will be executed if the requested search has found a single entry. If empty, Multiple flow will be executed
  • Empty flow - will be executed if the requested search didn't find anything. If empty, Multiple flow will be executed
